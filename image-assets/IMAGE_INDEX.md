## USAGE RECOMMANDÉ

### Pour vidéos Flow (background videos site)
→ Voir `FLOW_VIDEO_PROMPT.md` dans ce dossier

### Pour redesign UI
- Livres en format "Double" (vue face) = idéal pour cards produits
- Livres en format "Tranche" = idéal pour parallax/stack effect
- Images blog = idéal pour hero sections articles

### Pour animations Framer Motion
```tsx
// Exemple d'utilisation dans Next.js (CSS backgroundImage — pas besoin de remotePatterns)
<div style={{ backgroundImage: "url('https://tikoun-aolam.com/wp-content/uploads/2025/09/Otsar-Hayira-Fiche-Produit.jpg')" }} />

// Ou avec Next.js Image (remotePatterns requis dans next.config.ts — DÉJÀ CONFIGURÉ)
import Image from "next/image";
<Image src="https://tikoun-aolam.com/wp-content/uploads/..." fill className="object-cover" />
```
